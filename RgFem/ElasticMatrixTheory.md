# 线弹性材料弹性矩阵理论

## 1. 基本材料参数定义

对于线弹性材料，两个基本的拉梅常数定义为：

- 第一拉梅常数: `λ = (Eν)/((1+ν)(1-2ν))`
- 剪切模量（第二拉梅常数）: `μ = E/(2(1+ν))`

其中：
- `E`: 杨氏模量
- `ν`: 泊松比

也可以用杨氏模量和泊松比直接表示：
- `c = E/((1+ν)(1-2ν))`

## 2. 三维（3D）分析模式

### 2.1 本构关系

在三维问题中，应力-应变关系为：

```
⎡σxx⎤   ⎡D₁₁  D₁₂  D₁₂   0    0    0 ⎤ ⎡εxx⎤
⎢σyy⎥   ⎢D₁₂  D₁₁  D₁₂   0    0    0 ⎥ ⎢εyy⎥
⎢σzz⎥   ⎢D₁₂  D₁₂  D₁₁   0    0    0 ⎥ ⎢εzz⎥
⎢σxy⎥ = ⎢ 0    0    0   D₆₆   0    0 ⎥ ⎢γxy⎥
⎢σyz⎥   ⎢ 0    0    0    0   D₆₆   0 ⎥ ⎢γyz⎥
⎣σxz⎦   ⎣ 0    0    0    0    0   D₆₆⎦ ⎣γxz⎦
```

### 2.2 弹性矩阵系数

- `D₁₁ = c(1-ν) = E(1-ν)/((1+ν)(1-2ν))`
- `D₁₂ = cν = Eν/((1+ν)(1-2ν))`
- `D₆₆ = μ = E/(2(1+ν)) = c(1-2ν)/2`

## 3. 平面应力（Plane Stress）分析

### 3.1 假设条件

- `σzz = 0`（厚度方向应力为零）
- `εzz ≠ 0`（厚度方向应变存在）

### 3.2 本构关系

```
⎡σxx⎤   ⎡D₁₁  D₁₂   0 ⎤ ⎡εxx⎤
⎢σyy⎥ = ⎢D₁₂  D₁₁   0 ⎥ ⎢εyy⎥
⎣σxy⎦   ⎣ 0    0   D₃₃⎦ ⎣γxy⎦
```

### 3.3 弹性矩阵系数

- `D₁₁ = E/(1-ν²)`
- `D₁₂ = Eν/(1-ν²)`
- `D₃₃ = E/(2(1+ν)) = G`（剪切模量）

## 4. 平面应变（Plane Strain）分析

### 4.1 假设条件

- `εzz = 0`（厚度方向应变为零）
- `σzz ≠ 0`（厚度方向应力存在）

### 4.2 本构关系

```
⎡σxx⎤   ⎡D₁₁  D₁₂   0 ⎤ ⎡εxx⎤
⎢σyy⎥ = ⎢D₁₂  D₁₁   0 ⎥ ⎢εyy⎥
⎣σxy⎦   ⎣ 0    0   D₃₃⎦ ⎣γxy⎦
```

### 4.3 弹性矩阵系数

- `D₁₁ = c(1-ν) = E(1-ν)/((1+ν)(1-2ν))`
- `D₁₂ = cν/(1-ν) = Eν/((1+ν)(1-2ν)) × 1/(1-ν) = Eν/((1-ν²)(1-2ν)) × (1-ν)`
  简化为: `D₁₂ = Eν/((1+ν)(1-2ν))`
- `D₃₃ = E/(2(1+ν)) = G`（剪切模量）

## 5. 代码实现中的系数计算

### 5.1 三维情况
```cpp
const double c = m_E / ((1.0 + m_nu) * (1.0 - 2.0 * m_nu));
m_Ce[0][0] = c * (1.0 - m_nu);  // D₁₁
m_Ce[1][1] = c * (1.0 - m_nu);  // D₁₁
m_Ce[2][2] = c * (1.0 - m_nu);  // D₁₁
m_Ce[0][1] = c * m_nu;          // D₁₂
m_Ce[0][2] = c * m_nu;          // D₁₂
m_Ce[1][0] = c * m_nu;          // D₁₂
m_Ce[1][2] = c * m_nu;          // D₁₂
m_Ce[2][0] = c * m_nu;          // D₁₂
m_Ce[2][1] = c * m_nu;          // D₁₂
m_Ce[3][3] = m_E / (2.0 * (1.0 + m_nu));  // D₆₆
m_Ce[4][4] = m_E / (2.0 * (1.0 + m_nu));  // D₆₆
m_Ce[5][5] = m_E / (2.0 * (1.0 + m_nu));  // D₆₆
```

### 5.2 平面应力情况
```cpp
const double factor = m_E / (1.0 - m_nu * m_nu);
m_Ce[0][0] = factor;              // D₁₁
m_Ce[0][1] = factor * m_nu;       // D₁₂
m_Ce[1][0] = factor * m_nu;       // D₁₂
m_Ce[1][1] = factor;              // D₁₁
m_Ce[2][2] = m_E / (2.0 * (1.0 + m_nu));  // D₃₃
```

### 5.3 平面应变情况
```cpp
const double c = m_E * (1.0 - m_nu) / ((1.0 + m_nu) * (1.0 - 2.0 * m_nu));
m_Ce[0][0] = c;                             // D₁₁
m_Ce[0][1] = c * m_nu / (1.0 - m_nu);     // D₁₂
m_Ce[1][0] = c * m_nu / (1.0 - m_nu);     // D₁₂
m_Ce[1][1] = c;                             // D₁₁
m_Ce[2][2] = m_E / (2.0 * (1.0 + m_nu));  // D₃₃
```

## 6. 物理意义

- 弹性矩阵[D]将应变向量映射到应力向量：`{σ} = [D]{ε}`
- 矩阵的对称性反映了材料的弹性性质
- 不同的分析模式对应不同的物理假设，从而导致不同的本构关系
- 在有限元分析中，弹性矩阵用于计算单元刚度矩阵中的材料刚度贡献
